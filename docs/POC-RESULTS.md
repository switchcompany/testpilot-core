## Patterns Discovered During Run

The following patterns were auto-captured into `LEARNINGS.md`:

| # | Pattern | Impact |
|---|---------|--------|
| 1 | `@lru_cache` settings must call `.cache_clear()` in test fixtures | Prevented 12 test leaks |
| 2 | Mock where used (`app.pipeline.send_email`), not where defined | Fixed 8 import errors |
| 3 | `AsyncMock` required for async functions, `MagicMock` for sync | Fixed 6 test failures |
| 4 | `httpx.AsyncClient` + `ASGITransport` for FastAPI route testing | Enabled all API tests |
| 5 | Lifespan override needed to avoid HuggingFace model loading | Reduced test time by 90% |
| 6 | Real temp SQLite (not mocks) for database tests | Better test fidelity |
| 7 | `pytest.mark.asyncio` required on all async test functions | Fixed 15 test errors |
| 8 | Fallback behavior testing: verify each fallback path independently | Caught 3 bugs |
| 9 | Global `DB_PATH` swap for test isolation | Prevented data pollution |
| 10 | Pydantic model `.model_dump()` for assertion comparison | Cleaner assertions |
| 11 | `conftest.py` env vars must be set before module imports | Fixed config pollution |

## Sample Generated Test

```python
# test_pipeline.py — Generated by TestPilot Core

import pytest
from unittest.mock import AsyncMock, patch, MagicMock

@pytest.mark.asyncio
async def test_process_mention_escalation_detected():
    """When classifier returns ESCALATION, email should be sent and record saved."""
    mention = {
        "actor": {"name": "John Doe", "headline": "CEO at TechCorp"},
        "text": "This is the worst service I've ever experienced @BrandName",
        "permalink": "https://linkedin.com/posts/123",
        "platform": "linkedin"
    }

    with patch("app.pipeline.classify_text") as mock_classify, \
         patch("app.pipeline.send_escalation_email", new_callable=AsyncMock) as mock_email, \
         patch("app.pipeline.save_escalation", new_callable=AsyncMock) as mock_save:

        mock_classify.return_value = {
            "label": "ESCALATION",
            "confidence": 0.94,
            "keywords_found": ["worst"]
        }
        mock_email.return_value = True

        from app.pipeline import process_mention
        result = await process_mention(mention)

        assert result["classification"] == "ESCALATION"
        assert result["email_sent"] is True
        mock_email.assert_called_once()
        mock_save.assert_called_once()
```
